# 最具极客范的开源智能家居系统

> MyHome是由BLab大熊实验室开源的智能家居控制系统，包含软件硬件等诸多方面。致力于构建一套完全开源的智能家居解决方案，让一切能为你掌握，MyHome不会收集任何你的隐私数据保证你的隐私安全。
